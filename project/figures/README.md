# Figures

Historical figures generated for weekly meetings during the thesis research.

## Contents

Each `meeting_YY_MM_DD_*` folder contains figures presented at that week's meeting. These document the research progression from October 2023 through May 2024.

| Folder | Topic |
|--------|-------|
| `meeting_23_10_04_loglikelihood` | Log-likelihood optimization plots |
| `meeting_23_10_19_subpoisson` | Sub-Poissonian photon statistics |
| `meeting_23_10_26` | General progress |
| `meeting_23_11_03_coherence` | Coherence function analysis |
| `meeting_23_11_10_coherence` | Extended coherence analysis |
| `meeting_23_11_16_pixel_correlation` | Pixel correlation studies |
| `meeting_23_11_30_` | Coherence ratio analysis |
| `meeting_23_12_*` | December progress meetings |
| `meeting_24_*` | 2024 meetings |
| `report/` | Final thesis report figures |

## Note

These are preserved for reference. The `report/` subfolder contains the polished figures used in the final thesis.
