# Report

Jupyter notebooks used to generate figures and analysis for the thesis report.

## Notebooks

| Notebook | Description |
|----------|-------------|
| `introduction.ipynb` | Introduction chapter figures |
| `theory.ipynb` | Theoretical background visualizations |
| `method.ipynb` | Methodology illustrations and diagrams |
| `results.ipynb` | Results analysis and final figures |

## Usage

These notebooks were used to create publication-quality figures for the thesis. They serve as documentation of how each figure was generated and can be re-run to reproduce results.

```bash
jupyter notebook
# Then open the desired notebook
```

## Note

The notebooks may reference data files that have been moved to `archive/`. Update paths if you need to regenerate figures.
