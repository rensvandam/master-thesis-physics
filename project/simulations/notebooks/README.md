# Notebooks

Jupyter notebooks for visualization and analysis.

## Files

### `quick_visual.ipynb`
Quick visualization notebook for exploring simulation results interactively.

### `report_plots.ipynb`
Generates publication-quality figures for the thesis report.

## Usage

```bash
jupyter notebook
# Then open the desired notebook
```

Note: Notebooks may reference data paths that need updating if data has been moved to the archive.
